The <label> element

<label> is useful for screen-reader users. It also helps pointing radio buttons, you can click the <label> element to toggle it:



The for attribute of <label> should be equal to id atrribute of <input>.

The method attribute

The <form> tag admits an method attribute. The method can be "get" or "post".

Notes on get

  • Appends form data to the URL
  • NEVER use GET for sensistive data
  • URL limited to 2048 characters
  • Useful for bookmark results
  • Good for non-secure-data
