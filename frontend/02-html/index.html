What is HTML?

Hyper Text Markup Language.

  • Describes the structure of a Web Page.
  • Consist in a series of elements.
  • Each element is marked with a tag.

HTML Elements

Defined by a start tag, content and end tag:

‹tagname› Content. ‹/tagname›

HTML Links

‹a href="https://www.w3schools.com"› Link. ‹/a›

Link.

HTML Images

‹img src="https://www.w3schools.com/apple-touch-icon.png"›

HTML Attributes

  • HTML elements can have attributes.
  • always especified in the start tag.
  • usually name="value" pair.

src

Specifies the image path for an ‹img› tag.

‹img src="https://www.w3schools.com/apple-touch-icon.png"›

lang

Always include the language in the ‹html› tag.

‹html lang="es-AR"›

Width and height attributes

The ‹img› tag can contain width and height attributes, specifying image size in pixels.

‹img src="https://www.w3schools.com/apple-touch-icon.png" width="100" height="600"›

Alt attribute

Showed when img can't be displayed.

‹img href="./broken.link.png" alt="an awesome image with a broken link"›

Recommendations

  • Always quote attribute values: href="./link"
  • Use lowercase: src=""
  • Prefer doublequotes for HTML, single for Javascript
